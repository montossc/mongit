## Problem Statement

This task defines the visual contract for the app. The point is not superficial polish alone; it is to prevent ad hoc styling and make future product surfaces cheaper and more coherent to build.

## Scope

### In-Scope

Define and stabilize the visual token contract and core UI primitives needed to build mongit consistently. Scope includes semantic colors, spacing, typography, surface treatments, and foundational controls mapped to the product's 'premium calm' direction; it excludes a fully exhaustive component library.
